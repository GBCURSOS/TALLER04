Ejercicio 3: Medidas en CSS

Medidas Absolutas

Esta caja usa pixeles (px) para definir su ancho y alto. Es una medida absoluta y no depende del tamaño de la pantalla.

Esta caja usa centímetros (cm) para definir su ancho. Es una medida física, poco usada en web.

Esta caja usa milímetros (mm) para definir su ancho. Es una medida física, poco usada en web.

Esta caja usa pulgadas (in) para definir su ancho. Es una medida física, poco usada en web.

Esta caja usa puntos (pt) para definir el tamaño de fuente. 1pt = 1/72 de pulgada.

Esta caja usa picas (pc) para definir el tamaño de fuente. 1pc = 12pt.

Medidas Relativas

Esta caja usa em para definir el tamaño de fuente y el padding. 1em equivale al tamaño de fuente del elemento padre.

Esta caja usa rem para definir el tamaño de fuente y el padding. 1rem equivale al tamaño de fuente raíz (html).

Esta caja usa vw para definir su ancho. 1vw es el 1% del ancho de la ventana.

Esta caja usa vh para definir su alto. 1vh es el 1% del alto de la ventana.

Esta caja usa porcentaje (%) para definir su ancho respecto al contenedor.

Esta caja usa ex para definir el tamaño de fuente. 1ex equivale a la altura de la letra "x" en la fuente actual.

Esta caja usa ch para definir el ancho. 1ch equivale al ancho del carácter "0" en la fuente actual.
