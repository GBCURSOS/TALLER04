Ejercicio 4: Posiciones en CSS

Diferentes posiciones en CSS

position: static
Este div tiene posición estática. Es el valor por defecto y se posiciona según el flujo normal del documento. No se pueden usar top, right, bottom o left.
position: relative
Este div tiene posición relativa. Se posiciona respecto a su posición original y puede moverse usando top, right, bottom o left.
position: absolute
Este div tiene posición absoluta. Se posiciona respecto al ancestro posicionado más cercano o al body si no hay ninguno.
position: fixed
Este div tiene posición fija. Permanece en la misma posición respecto a la ventana del navegador, incluso al hacer scroll.
position: sticky
Este div tiene posición sticky. Se comporta como relativa hasta que se alcanza un punto de desplazamiento, luego se comporta como fija.
